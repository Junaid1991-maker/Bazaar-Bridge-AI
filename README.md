🌉 Bazaar-Bridge: Headless AI Inventory EngineBazaar-Bridge is a lightweight, agentic pipeline designed to digitize wholesale order management in traditional markets (like Karachi's Jodia Bazaar). It specializes in handling "Messy" Roman Urdu/English inputs and converting them into professional business leads.
🚀 The ChallengeWholesalers often operate in fast-paced, high-noise environments using Roman Urdu (e.g., typing "Lal Mirhc" instead of "Red Chili"). Most LLMs are too heavy to run locally on mid-range hardware, and official WhatsApp APIs are cost-prohibitive for small vendors.
🛠️ The "Headless" SolutionThis project uses a Tiered Logic Architecture to stay fast and secure on a Dell Latitude 5490 (16GB RAM):Local Matcher (The Filter): Uses RapidFuzz to correct typos locally, saving API costs.Secure Ledger (The Vault): An SQLite database with parameterized queries to prevent SQL injection.Cloud Brain (The Agent): Gemini 1.5 Flash handles cultural nuances and Roman Urdu greeting generation.Universal Bridge: Generates encrypted WhatsApp links for instant customer notification.
